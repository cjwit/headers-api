Timestamper

Usage:

Add a string containing either a natural language date () or UNIX timestamp to this application's URL.

Example inputs:

Unix time (milliseconds since January 1st, 1970

  • https://api-projects-cjwit.c9users.io/378518400

Other date strings (spaces work, as does %20):

  • https://api-projects-cjwit.c9users.io/December 30, 1981
  • https://api-projects-cjwit.c9users.io/December%2030,%201981
  • https://api-projects-cjwit.c9users.io/30 December, 1981
  • https://api-projects-cjwit.c9users.io/30 Dec 1981
  • https://api-projects-cjwit.c9users.io/12 30 1981
  • https://api-projects-cjwit.c9users.io/12-30-1981
