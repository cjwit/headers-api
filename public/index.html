Timestamper

Usage:

Add a string containing either a natural language date () or UNIX timestamp to this application's URL.

Example inputs:

Get the current date:

  • http://polar-cliffs-58838.herokuapp.com/today

Unix time (milliseconds since January 1st, 1970

  • http://polar-cliffs-58838.herokuapp.com/378518400

Other date strings (spaces work, as does %20):

  • http://polar-cliffs-58838.herokuapp.com/December 30, 1981
  • http://polar-cliffs-58838.herokuapp.com/December%2030,%201981
  • http://polar-cliffs-58838.herokuapp.com/30 December, 1981
  • http://polar-cliffs-58838.herokuapp.com/30 Dec 1981
  • http://polar-cliffs-58838.herokuapp.com/12 30 1981
  • http://polar-cliffs-58838.herokuapp.com/12-30-1981
